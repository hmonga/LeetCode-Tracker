# LeetCode Tracker

A web application to track your LeetCode progress and statistics. Users can sign up, enter their LeetCode username, and view their solved problems, acceptance rate, and streak.

## Features

- User authentication with Firebase
- Track LeetCode statistics (total solved, easy/medium/hard breakdown)
- View acceptance rate
- Streak tracking
- Progress visualization with charts

## Tech Stack

Frontend: React, Vite, Firebase, Axios
Backend: Spring Boot, Java 17, Maven
